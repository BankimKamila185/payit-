# 🛡️ UPI Fraud Shield — Full Feature List

## A. Detection Signals (guard kya-kya dekhta hai)
1. **Behavioral analysis** — amount normal se kitna bada, ajeeb time (raat), spending pattern
2. **Device fingerprinting** — naya/anjaan phone se transaction
3. **Receiver risk** — receiver account naya hai / kal bana hai / pehle se flagged
4. **Velocity detection** — jaldi-jaldi bahut saare transfers (bot/panic pattern)
5. **Graph mule-chain detection** ⭐ — paisa A→B→C→D ghoom raha (money laundering ring)

## B. Engine Capabilities (dimaag ki taakat)
6. **Real-time scoring** — har transaction <200ms mein score (0-100)
7. **3-tier smart decision** — Safe / Review (extra verify) / Block — *false positives se bachne ke liye*
8. **Explainability / reason codes** — har flag ke saath plain-English "kyun" (black box nahi)
9. **Self-learning ready** — confirmed cases se behaviour update (future)

## C. Product / Demo Features (jo screen pe dikhega)
10. **Live dashboard** — transactions ka real-time feed, 🔴/🟢 flags, ms counter
11. **Attack simulator** — "Launch Fraud Attack" button, live mule chain pakadta
12. **Graph visualization** — mule chain screen pe chamakti hui (dots + arrows)
13. **Real test-mode payment block** — Razorpay test payment ko live block/allow

## D. Investigation Features (tera idea — fraud ke baad)
14. **Fraud reporting** — victim report kare "yeh payment fraud tha"
15. **Chain tracing** — poori mule chain trace karke dikhana
16. **Mule report generation** — flagged accounts ki ready report bank/police ke liye
17. **Freeze recommendation** — "yeh accounts turant freeze karo"

## E. Pitch / Credibility
18. **RBI MuleHunter.AI + DPIP alignment** — real regulatory direction
19. **₹36,000 cr problem** — macro impact number
